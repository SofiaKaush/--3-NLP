Проект ИИ-ассистентов на базе Flask (GPT-4o mini)

Данный репозиторий содержит серию учебных приложений, демонстрирующих эволюцию интеграции больших языковых моделей (LLM) в веб-интерфейсы.

Состав проекта

Проект разделен на 4 этапа развития функционала:

app_v1.py (Текстовый чат): Базовая реализация диалога с сохранением истории контекста.

app_v2.py (Зрение / Vision): Добавлена возможность загрузки и анализа изображений.

app_v3.py (Генерация изображений): Реализован отдельный механизм для создания картинок по текстовому описанию с помощью внешней кнопки.

app_v4.py (Интеллектуальный агент): Использование технологии Function Calling. Модель сама решает, когда ей нужно ответить текстом, а когда вызвать инструменты (рисование или погоду).

Технические детали

Backend: Python (Flask).

AI API: OpenRouter (совместим с OpenAI SDK).

Модель: gpt-4o-mini.

Интерфейс: Чистый HTML/JavaScript без использования сторонних иконок и тяжелых библиотек.

Инструкция по запуску

Для запуска любого из приложений выполните следующие шаги:

Установите необходимые библиотеки:

pip install flask openai


Установите API-ключ в переменные окружения:

export OPENROUTER_API_KEY="ваш_ключ"


Запустите файл приложения:

python app_v4.py


Откройте браузер по адресу, указанному в терминале (обычно http://127.0.0.1:5004).

Особенности реализации

В приложениях реализована возможность ввода или изменения API-ключа прямо через веб-интерфейс, что удобно для тестирования без перезапуска сервера. Для генерации изображений в бесплатных режимах используется сервис Pollinations.ai.
